# BusinessGraph Pro

## Мүмкіндіктері:
- Бірнеше өнім енгізу
- Айнымалы шығындарды жазу
- Тұрақты шығынды көрсету
- Залалсыздық нүктесін есептеу
- Түсім мен шығын функцияларының графигін салу
- Қазақша және Орысша интерфейс
- Автор: Арал ауданы №71 мектеп
